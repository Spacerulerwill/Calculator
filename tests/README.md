# Unit Tests

All valid unit tests are based on the following mathematical expressions.
```
1) 3 + 5 = 8
2) 7 - 2 = 5
3) 4 * 6 = 24
4) 15 / 3 = 5
5) 2 * (4 + 7) = 22
6) (10 - 3) + (8 / 2) = 11
7) 6 * (4 - 2) + 5 = 17
8) (15 / 4) * (2 + 1) = 9
9) (7 - 2) * (4 + 3) - 10 = 25
10) 4 * ((6 / 2) + 1) - 5 = 11
```

Currently there are no unit tests for testing invalid expressions.
